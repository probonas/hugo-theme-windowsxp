Welcome to my blog! This is a sample post to demonstrate the Windows XP Hugo theme.

## What is this?

This Hugo theme recreates the **Windows XP Luna Blue** desktop experience in your browser. On desktop, you get:

- A fully functional **taskbar** with a start menu
- **Draggable, resizable windows** with minimize/maximize/close buttons
- An **Explorer-style** file browser for blog posts and projects
- A **Windows Picture and Fax Viewer** for image galleries

On mobile, the theme transforms into a **Windows Mobile 2003** Pocket PC interface with a today screen, navigation bar, and command bar.

    fmt.Println("Hello from Windows XP!")
